This game was initially made for the spring 2026 OSU VGDC game jam by myself and Nikole Ewing
I did all of the programming and work within Godot while Nikole drew all of the art.
A playable version of the game is on Itch.io at https://wafflesthehutt.itch.io/reboot
We are continuing to develop Reboot. Our current goal is to develop ~10 levels. With 3 zones, 3 levels per zone, and one tutorial level. 
The exact number of levels is subject to change though depending on how the levels end up being structured
